\# SCHEDULR



SCHEDULR is a logic-based scheduling system that uses \*\*Prolog and Node.js\*\* to generate optimized schedules.



\## Features

\- Logic-based scheduling

\- Conflict-free timetable generation

\- Prolog rule engine

\- Node.js backend

\- Simple web interface



\## Tech Stack

\- Prolog

\- Node.js

\- HTML

\- CSS

\- JavaScript



\## How It Works

1\. User inputs tasks.

2\. Node.js processes the request.

3\. Prolog applies scheduling rules.

4\. Optimized schedule is generated.
